## Features

- **Dark Mode**
- **Build with TailwindCSS**: 
- **Responsive**


## Getting Started

1. Clone the repository:
```bash
# Clone the repository
git clone https://github.com/eAntillon/void-astro.git
cd void-astro
#  Install dependencies
npm i
#  Start the development server
npm run dev
```
